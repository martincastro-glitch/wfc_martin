# Wave funtion collapse

Este repositorio contiene el código de la herramienta de generación de imágenes por medio del algoritmo de colapso de la función de onda. Está pensado para que los estudiantes de la materia de Diseño de Personajes puedan sustituir las imágenes y las reglas de la herramienta para generar imágenes nuevas.

## Procedimiento

### Instalación

1. Asegúrate de tener instalado Git y Gitkraken (necesitas contar con una cuenta de GitHub).
2. Descarga e instala un editor de código como Visual Studio Code o Antigravity.
3. Instala los complementos de Live Server y Edit CSV.

### Edición de reglas

1. Reemplaza las imágenes de la carpeta "azulejos" por las imágenes que diseñaste.
2. Edita el archivo reglas.csv conforme a las imágenes que diseñaste teniendo en cuenta el ID de cada una.
3. Haz pruebas de usando live server. 

### Publicación

1. Empuja los cambios de tu repositorio a tu cuenta de GitHub.
2. Ve la configuración de tu repositorio en GitHub y ve a la sección de Pages.
3. Configura la página para que se genere desde la rama main y espera a que se genere la página y se muestre la URL.
